# Enscape Suite — Portable Real‑Time Rendering Environment

## About Enscape Suite

Enscape Suite provides a complete real‑time rendering and virtual reality visualization environment directly integrated into leading CAD software. Enscape Suite combines live walkthrough navigation, material editing with physically‑based rendering (PBR), lighting adjustment (sun, artificial, IES profiles), and asset placement (vegetation, entourage, vehicles, furniture). Enscape Suite supports integrated plugins for Revit, SketchUp, Rhino, Archicad, Vectorworks, and custom CAD, with one‑click 360° panorama generation (equirectangular, stereo, VR headset ready), BIM data visualization (dynamic property display, filter by metadata (fire rating, acoustic class, manufacturer, cost, installation date), export schedules, colour‑coded views by parameter value (energy performance, U‑value, load‑bearing capacity), clash detection heatmaps, and design review collaborative markup. Enscape Suite enables users to export standalone executable (web standalone, desktop .exe with navigation, VR), video (MP4, 8K resolution, 60fps), panorama (web share, embed), batch render (multiple views, scene states), and documentation (image, PDF report, BCF comments). Enscape Suite runs on standard business workstations. Enscape Suite is trusted by architects, interior designers, landscape architects, urban planners, BIM managers, and real estate developers worldwide.
